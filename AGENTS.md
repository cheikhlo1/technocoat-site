# Instructions pour Codex — Projet Technocoat

Ce projet est une application web de gestion de production.

## Technologies autorisées
- HTML
- CSS
- JavaScript pur

## Technologies interdites
- React
- Vue
- Angular
- Bootstrap
- Backend
- Base de données externe

## Fichiers principaux
- index.html
- style.css
- script.js

## Règles
- Ne pas supprimer les sections existantes.
- Ne pas changer le nom des fichiers principaux.
- Garder une interface professionnelle, claire et responsive.
- Faire des modifications progressives.
- Résumer chaque modification à la fin.
- Ne jamais ajouter de données confidentielles.
